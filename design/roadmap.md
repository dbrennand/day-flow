# DayFlow — Product Roadmap

This document captures feature ideas and enhancements that are out of scope for the initial release (V1) but worth revisiting as the app matures.

---

## Post-V1 Considerations

- **Template Day** — Quickly load a pre-set routine so the user doesn't have to rebuild a typical day from scratch each morning
- **Ambient Sound Options** — Optional, subtle focus audio to help with concentration during tasks
- **Recurring Daily Tasks** — The ability to mark a task as recurring so it automatically appears each day without needing to be re-entered
- **Mobile / Phone Optimised Layout** — A dedicated layout for smaller phone screens, beyond the current tablet and desktop responsive design
- **Export / Print View** — A clean, printable or exportable summary of the day's plan
